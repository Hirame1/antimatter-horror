# Antimatter Dimensions: Cosmic Horror
### 反物質の深淵へ — コズミックホラー

> *"When we seek growth, consider the presence that watches it grow."*
> 「成長を求めるとき、その成長を見守っている存在のことを忘れてはならない」

---

## Overview / 概要

**Antimatter Dimensions: Cosmic Horror** is an interactive horror story told in four arcs, presented as a single-page web experience.

It reimagines the browser/mobile idle game *Antimatter Dimensions* as a piece of cosmic horror fiction — asking the question: *what if the numbers you were generating were being extracted from something vast, ancient, and aware?*

**Antimatter Dimensions: Cosmic Horror** は、ブラウザ/モバイルゲーム「Antimatter Dimensions」の世界観を宇宙的ホラー小説として再解釈した、4部構成のインタラクティブな読み物です。

「あなたが積み上げていた数字は、広大で、古く、意識を持つ何かから引き抜かれていたとしたら？」という問いから始まる物語です。

---

## Story Structure / 物語の構成

| Arc | Title (EN) | タイトル (JA) | Game Elements |
|-----|-----------|--------------|---------------|
| I | First Infinity | 最初の無限 | Dimensions 1–8, Tickspeed, Dimensional Sacrifice |
| II | Eternity | 永劫回帰 | Infinity Points, Replicanti, Infinity Dimensions, Eternity |
| III | Reality | 現実機械 | Reality Machine, Glyphs, Perks, Black Hole |
| IV | Celestials | 星神たちの審判 | Teresa, Effarig, The Nameless Ones, V, Ra, Lai'tela, Pelle |

The story follows **Haruka Mishima**, a researcher who discovers that the antimatter her facility is generating is not being created — it is being *extracted* from somewhere else.

物語の主人公は研究員・**三島遥**。施設で生成されていると思っていた反物質が、実はどこか別の場所から「引き抜かれていた」ことに気づく彼女の軌跡を追います。

## Features / 機能

- **Bilingual** — Full Japanese and English text, switchable at any time
- **Font size control** — Small / Medium / Large
- **Tab navigation** — Switch between arcs freely
- **Animated UI** — Starfield background, floating dimension numbers, reveal animations, arc-specific color themes
- **Mobile-friendly** — Responsive layout, Safari-compatible

---

- **日英バイリンガル** — 全文を日本語・英語で収録。いつでも切り替え可能
- **文字サイズ変更** — 小・中・大の3段階
- **タブナビゲーション** — 各章を自由に行き来可能
- **アニメーションUI** — 星空背景、浮遊する次元数値、スクロール連動アニメーション
- **モバイル対応** — レスポンシブレイアウト、iOS Safari 対応

## How to Read / 読み方

1. Open `index.html` (or visit the GitHub Pages URL) in any modern browser
2. Use the **tab bar at the bottom** to navigate between arcs
3. Use the **⚙ gear button** (top right) to switch language or adjust font size

---

1. 任意のモダンブラウザで `index.html` を開く（またはGitHub Pages URLにアクセス）
2. **画面下部のタブバー**で章を切り替える
3. **右上の ⚙ ボタン**で言語切り替えとフォントサイズ変更

## About the Creation / 制作について

This project was created in a conversational session with **[Claude](https://claude.ai)** (Anthropic), a large language model AI assistant.

The story concept, narrative direction, game-to-horror mapping, and creative decisions were provided by the human author. The majority of the writing — both Japanese and English prose, HTML/CSS/JS code, and visual design — was generated and iterated upon through dialogue with Claude.

このプロジェクトは、AnthropicのAIアシスタント **[Claude](https://claude.ai)** との対話形式のセッションで制作されました。

ストーリーのコンセプト、物語の方向性、ゲーム要素のホラーへの落とし込み、クリエイティブな判断は人間の作者が担当しました。日本語・英語の散文、HTML/CSS/JS のコード、ビジュアルデザインの大部分は、Claudeとの対話を通じて生成・反復されています。
